### Basic app | users

##### Server

-   `cd users-server`
-   `yarn install`
-   `yarn dev`
-   open http://localhost:4000/graphql

##### Client

-   `cd users-client`
-   `yarn install`
-   `yarn dev`
-   open http://localhost:1234
